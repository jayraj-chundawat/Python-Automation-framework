
# Python API Automation Framework

This framework is currently under development to automate REST API testing using Python and Pytest.

## Project Structure (in progress)

- `api/` – Modules for API clients and services
- `config/` – Configuration files
- `utils/` – Helper functions and utilities
- `tests/` – Test cases using Pytest
- `schemas/` – JSON schema files for response validation
- `testdata/` – Test payload inputs
- `reports/` – Test report output folder
